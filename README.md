# Little Lemon Capstone Project

A React-based table reservation application for Little Lemon Restaurant.

## Features

* Table booking form
* Dynamic available times
* Form validation
* Booking confirmation page
* Reservation history
* Local Storage persistence
* Responsive design
* Accessibility improvements
* Unit tests

## Installation

Clone the repository:

git clone https://github.com/your-username/your-repository.git

Install dependencies:

npm install

Run the project:

npm start

Run tests:

npm test

## Technologies Used

* React
* React Router
* JavaScript
* CSS
* React Testing Library
* Jest
